# Harry Lewis — Strength & Conditioning Specialist

Single-page landing site for Harry Lewis, MSc Strength & Conditioning Specialist, running small group personal training sessions at Elite Physical Medicine, Aylesbury.

## Stack

- HTML5
- CSS3 (no frameworks)
- Vanilla JavaScript

## Structure

```
/
├── index.html        # Single page, all 7 sections
├── styles.css        # All styles
├── main.js           # Navigation behaviour and smooth scroll
├── images/
│   ├── hero.jpg      # Squat photo in clinic (hero background)
│   ├── about.jpg     # Arms crossed headshot
│   ├── laptop.jpg    # Programme on screen
│   └── handshake.jpg # Handshake photo in clinic
└── README.md
```

## Sections

1. **Hero** — Full viewport, background image, headline and CTAs
2. **Credentials strip** — Four key facts inline
3. **How it works** — Three numbered steps with laptop image
4. **Pricing** — Two plans: £135/mo and £179/mo
5. **About** — Bio, credential pills, headshot and handshake photo
6. **Book** — Calendly embed (https://calendly.com/harrylewishtl/30min)
7. **Footer** — Contact details

## Deploy to Vercel

1. Push this repository to GitHub
2. Go to [vercel.com](https://vercel.com) and import the repository
3. No build step required — Vercel serves `index.html` directly as a static site
4. (Optional) Add a custom domain in Vercel project settings
